# chicago-lead

how much lead is pouring out of chicago's fixtures?

## requirements
* csvkit (`pip install csvkit`)
* pyesridump (`pip install pyesridump`)

## to reproduce

clone this repo and from a terminal window, run:

`make all`

the files should land in `output/`.
